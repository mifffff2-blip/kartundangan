Panduan Deploy
Langkah lengkap dari generate hingga link sampai ke tangan klien.
🚀 Alur Kerja per Order
1
Klien order — catat nama, tema, tanggal, lokasi, rekening via WhatsApp atau form order.
2
Minta aset — foto prewedding (maks. 5, format JPG/PNG) + lagu MP3 (opsional).
3
Buat undangan — isi form di halaman ini, upload foto & lagu, klik Generate.
4
Deploy ke Netlify — drag & drop file hasil generate, catat URL Netlify.
5
Tambah subdomain — di Niagahoster DNS, tambah CNAME: nama-klien → URL Netlify.
6
Kirim ke klien — link siap: nama-klien.kartundangan.id 🎉
Estimasi waktu pengerjaan per order: 15–30 menit
Kapasitas per bulan: Tidak terbatas (Netlify gratis)
Biaya hosting per undangan: Rp 0
